# Ze Chen - Personal Portfolio

Personal portfolio website showcasing my work in DeFi, product management, and software engineering.

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router
- GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
